3 ОПИС ВИКОНАНОЇ РОБОТИ


	У процесі виконання практичної роботи було досліджено програмну систему Notion, призначенням якої є управління знаннями, нотатками, базами даних та спільною роботою. Система поєднує в собі текстовий редактор, систему організації даних та інтеграції з іншими сервісами, що робить її популярною серед користувачів різного рівня. 
	Notion має клієнт-серверну архітектуру з розподіленою обробкою даних. Основними компонентами є:
1.	Клієнтські додатки (веб, мобільний, десктопний).
2.	Серверна частина, що обробляє запити.
3.	База даних для зберігання контенту.
4.	Система реального часу для синхронізації даних.
Основною базою для зберігання даних Notion є PostgreSQL, а для кешування використовується Redis. В свою чергу, для для зберігання файлів та медіаконтенту використовується AWS S3.
Загалом було зроблено висновок, що програмна система Notion використовує сучасні технології та гнучку архітектуру для надання якісного сервісу. Завдяки мікросервісам, ефективному зберіганню даних та підтримці реального часу, вона є потужним інструментом для організації інформації та командної роботи.
